a yeoman generator for kissy pie

## install
### 安装yeoman
````sh
npm install yo
````

### 安装Generator

第一步:

````sh
git clone git@github.com:maxbbn/generator-kissy-pie.git
cd generator-kissy-pie
npm link
````

第二步, 在你的应用目录内:

````sh
npm install generator-kissy-pie  #安装本地版本
yo kissy-pie #执行yeoman
````
